Notes

The table for the two-qubit system and the patterns in it have been taken from M.Waegell and P.K.Aravind, J Phys A: Math. Theor. 44, 505303 (2011).

P.Lisonek, R.Raussendorf and V.Singh. arXiv:1401.3035, showed that the two-qubit system has 264 “parity proofs” in it (the technical term for the patterns shown here). Only a few of them can be seen using the menu.
